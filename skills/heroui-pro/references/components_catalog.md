## 组件结构

每个组件目录包含：
- `js/` - JavaScript 实现
  - `app.js` - 主入口文件
  - 其他支持文件（如 `sidebar.js`, `message-card.js` 等）
- `ts/` - TypeScript 实现
  - `app.tsx` - 主入口文件
  - 其他支持文件（如 `sidebar.tsx`, `message-card.tsx` 等）

## 常用依赖

所有组件依赖以下包：
- `@heroui/react` - HeroUI 核心组件库
- `@iconify/react` - 图标库
- `react` - React 框架

部分组件还使用：
- `@heroui/use-clipboard` - 剪贴板工具
- Next.js 相关包（用于路由和客户端组件）
